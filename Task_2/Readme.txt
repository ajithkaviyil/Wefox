This piece of code is written in python language, hence you need to have python installed in your local machine.
Install requests library of python to the local machine using the command pip install requests.
This code covers the 3 scenarios which include:
	- Creation of resource
	- Updation of resource
	- Deletion of resource
All the above sceanarios are covered under different functions, hence the required function can be called passing function arguments.
FQDN argument in the function refers towards the website, ie., petstore3.swagger.io in this case
ID argument is the reference of the ID of the pet to be created or already created.
name refers to the name of the pet.
